Folder or File Naming
    FolderNaming:
    --> CamelCase with starting letter capital.
    --> Ex:coding grad, CODINGGRAD, Coding Grad --> Unacceptable
    --> Ex: CodingGrad, FullStackDataScience(FDS)
    FileNaming:
    --> All smallcase letters
    --> Snake_case
    --> introduction_course
    --> try to not use prepositions, article, conjuctions 

    FileSystem:
    --> Its a software which will map your paths to address on disk
    --> FTS, NFTS -- File Tables
    --> Paths:
        1. Absolute Path -> Full path -> c:\Users\Vamshi\CodingGrad\start.txt
            --> Root Folder(Starting Folder) --> /(Linux), c:Drive(Windows)
            --> Os Folders --> /usr, /bin, /lib, /sbin, /home, /mnt
            --> Home Folder --> /home/sai (linux), c:\Users\sai(windows)
            --> Command Prompt / Terminal --> Default --> Home Folder

        2. Relative Path --> .\CodingGrad\sample.txt
            --> pwd --> Linux Command --> Present Working Directory
            --> w.r.t PWD where is the path --> relative path
